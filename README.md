# ✈ Flight Management System

## 📌 Project Overview

Flight Management System is a web application developed to manage flight bookings, passenger details, and scheduling operations efficiently.

## 🚀 Technology Used

Frontend:
• Angular  
• TypeScript  
• HTML  
• CSS  

Backend (if applicable):
• .NET Core API  

Database:
• SQL Server  

Tools:
• Git  
• VS Code  

---

## ⭐ Features

✔ Flight listing  
✔ Passenger management  
✔ Booking system  
✔ Search flights  
✔ Responsive UI  
✔ Error handling  

---

## 🏗 Architecture

Frontend → Angular UI  
Backend → REST API  
Database → SQL Server  

3 Tier Architecture:
Controller → Service → Data Layer
